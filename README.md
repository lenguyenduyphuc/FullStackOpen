## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm/yarn
- Git

### Installation

```bash
# Clone repository
git clone https://github.com/your-username/fullstackopen.git

# Navigate to project
cd fullstackopen

# Install dependencies
npm install

# Start development server
npm start
```
